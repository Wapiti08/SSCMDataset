## LimaCharlie

## Features
- cloud-based setting up, isolated environment
- multiple-sensors, cover process, network, system, file, module, syslog, dns ...
- observe, search, detect

## Instructions
- sign up for LimaCharlie

    https://app.limacharlie.io/signup

- add organization

- add sensors and install them locally

- create sigma rules according to pre-configured attacks

- output results and original events to storage bucket
